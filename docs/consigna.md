# Trabajo Práctico 1 — Catch Ball

## Consigna

Hacer un videojuego en 2D estilo **"Catch Ball"** con lo visto en la cursada.

Se deberán cumplir ciertas condiciones para estar en un determinado "tier".

---

## Tiers

### Bronce (1 - 4)

- Main Menu y Game Scene.
- Distintos objetos que caen que dan diferentes puntos.
- Condición de victoria y derrota.
- Dificultad progresiva.
- UI animada.
- Página de Itch.io (base).

### Plata (5 - 8)

- Power Ups.
- Game juice (partículas y screen shake).
- Highscore.
- Página de Itch.io con screenshots y descripción del juego.

### Oro (8 - 10)

- Splash Screen y Options Scene.
- Mecánica innovadora.
